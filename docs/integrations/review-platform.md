# Review Platform Integration (Schema-Safe)

## Goal

Display real, visible review excerpts on-site and keep schema policy-compliant.

## Provider Scaffold

- `REVIEW_PLATFORM_PROVIDER`
- `REVIEW_PLATFORM_PLACE_ID`
- `REVIEW_PLATFORM_API_KEY`

## Rules

- Only use reviews that are visibly rendered on-page.
- Do not inject fake or unverifiable aggregate ratings.
- Keep author, source, and date visible when implemented.
