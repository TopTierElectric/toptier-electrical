# On-Page Enhancement Template

## Title formula

`{Primary Service/Intent} in {Location} | {Trust Signal}`

## Meta description formula

`Need {service}? {Brand} provides {top 2 value props} in {location}. {CTA}.`

## Structural outline

1. H1 intent match
2. Intro (problem + outcome)
3. H2 services/use cases
4. H2 process/timeline
5. H2 pricing factors (if transactional)
6. H2 FAQs (only if helpful)
7. Trust section (license, owner/operator, service area)
8. CTA section

## Internal links

- 2–4 contextual links to canonical supporting pages.
- Anchor text should map to ownership cluster terms.
